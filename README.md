# PureSpace Commercial Cleaning Website

Live website: https://purespace.space

This is the official website for **PureSpace Commercial Cleaning Co.**, a premium commercial cleaning service.

The website was built with a focus on:
- Clean, modern design
- Professional business presentation
- Strong call-to-action for real clients
- Smooth animations and polished UI

---

## About the Project

This is a real production website built for a real business.
It is not a template or demo project.

The goal of the website is to:
- Present the brand professionally
- Attract commercial clients
- Make booking and contact simple
- Look premium on desktop and mobile
